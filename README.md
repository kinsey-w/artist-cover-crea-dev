# Creative Dev - Site vitrine & canvas

Ce projet est un **site vitrine** créé pour accompagner la sortie d’un **album live** de *The Weeknd*.
L’objectif est de proposer une **jaquette d’album générative**, que l’utilisateur peut personnaliser tout en respectant l’univers visuel de l’artiste.

## Concept & Moodboard

![Moodboard](/public/moodboard.png)

La DA repose sur un **moodboard** composé de :

* visuels de concerts
* danseurs masqués en costumes rouges
* lumières rouges, violettes et bleues

Ces références ont guidé les choix graphiques :

* **pattern de danseurs** génératif pour représenter la scène
* **couleurs contrastées** contrôlables par l’utilisateur
* **logo et texte** placés en haut
* **artiste en bas de l’image**


## Lancer le projet 

1. Installer les dépendances :

```bash
npm install
```

2. Lancer le serveur :

```bash
npm run dev
```

3. Ouvrir le projet dans le navigateur :  
   Vite affiche une URL (généralement `http://localhost:5173`) à ouvrir dans le navigateur.
